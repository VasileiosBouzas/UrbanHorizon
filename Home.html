Project Definition

The Urban Horizon project is set up in order to provide a user-friendly portal that will provide users with Skyview Factor (SVF) values at requested coordinates. The SVF is typically represented by a dimensionless value between 0 and 1, where 0 indicates the sky is completely obstructed by obstacles and 1 indicates there are no obstructions (Brown et al., 2001).

The Urban Horizon project is set up by TU Delft students from the MSc programme Geomatics for the Built Environment. The team consists of the students Vasileios Bouzas, Rob de Groot, Melika Sajadian, Nikolaos Tzounakos and Teng Wu. The project is part of the Synthesis Project which is a culmination of acquired skills in the first year of the MSc programme.
